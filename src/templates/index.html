API Documentation MC - Mochila Champions

Endpoint

/decklist

Parameters

  • ydke: (string, required) The encrypted deck link in ydke:// format. This should be included as a query parameter.

Response

  • arquetype: "generic" - Indicates the archetype of the deck. In this case, "generic" means that the deck does not belong to any specific archetype.
  • ydke: "ydke://..." - Deck code in YDKE format, used to export/import decks on supported platforms.
  • decklist:
    • cod:
      • main: [86377375, 86377375] - IDs of cards in the main deck.
      • extra: [] - IDs of the cards in the extra deck. Here it is empty, indicating that there are no extra cards.
      • side: [102380, 8267140, 8267140] - IDs of cards in the side deck.
    • deck:
      • main:
        • card_images: URLs of the images of the letter. Includes:
          • image_url: - High resolution image.
          • image_url_small: - Low resolution image.
          • image_url_cropped: - Cropped image.
        • desc: "You can discard this card; ..." - Description of the card's effect.
        • frameType: "effect" - Type of card frame. In this case, it is an effect card.
        • humanReadableCardType: "Effect Monster" - Letter type in human readable format.
        • id: 86377375 - Unique identifier of the letter.
        • name: "Ahrima, the Wicked Warden" - Name of the letter.
        • race: "Fiend" - Monster race.
        • type: "Effect Monster" - Type of letter.
        • ygoprodeck_url:- Link for more information about the letter.
      • extra: [] - Empty list, indicating no extra cards.
      • side: [] - Contains the same structure for the main deck, but here it is empty for simplicity
  • full_cards:
    • deck: 48 - Total cards in the deck (main + extra + side).
    • main: 40 - Total cards in the main deck.
    • extra: 0 - Total cards in the extra deck.
    • side: 8 - Total number of cards in the side deck.
    • arquetype: 0 - Total number of cards of the archetype in the deck.
    • generic: 48 - Total generic cards in the deck.
    • arquetypes_invalidos: 0 - Total invalid cards of the archetype.
  • number_copy_cards:
    • monster_main: 19 - Number of monsters in the main deck.
    • spell_main: 8 - Number of spells in the main deck.
    • trap_main: 13 - Number of traps in the main deck.
    • banlist: 11 - Number of cards affected by the ban list.
  • cards_invalids:
    • cod_arquetype: [] - Invalid cards from other archetypes.
    • cod_banlist: [] - Cards invalid by ban list.
  • cards_banlist_no_deck:
    • cod_condition: ["1475311:3", "4931121:1"] - List of card IDs and the maximum amount allowed by the banlist.
